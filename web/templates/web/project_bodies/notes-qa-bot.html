A Streamlit RAG app: you upload or paste notes, embed them with a local Hugging Face model, store vectors in Chroma, and ask grounded questions using LangChain. Groq powers the chat model.

Flow: chunk notes → local embeddings → Chroma index → retriever + QA chain for answers with optional source expanders.

Demo

Full app.py (LangChain, Chroma, Groq HTTP clients, TLS) lives in the project repository.
