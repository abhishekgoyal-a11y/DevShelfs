Open-source Model Landscape

Llama 3, Mistral, Gemma, Phi, and Qwen — what each model family is good at, their size tradeoffs, and which to pick for different tasks.

Series progress3 / 9

Llama 3 — Meta

The most widely used open-source model family. Strong general-purpose performance, large community, and deep integration across tools and frameworks.

ModelSizeBest for
llama3.2:1b~1 GBEdge devices, simple extraction
llama3.2:3b~2 GBEveryday tasks on 8 GB RAM
llama3.1:8b~5 GBStrong all-rounder, 128k context
llama3.1:70b~40 GBGPT-4 class, needs 64 GB RAM

Llama 3.1 8B is the best starting point for most developers — it balances quality, speed, and RAM requirements.

Mistral — Mistral AI

Known for punching above its weight class. Mistral 7B performs on par with Llama 13B on many benchmarks. Fast and efficient — a great default for 8 GB machines.

ModelNotes
mistral:7bFast, efficient. Best 7B for instruction following
mistral-nemo:12b12B, 128k context window — great for long documents
mixtral:8x7bMixture-of-experts, ~26 GB, near GPT-3.5 quality

Gemma — Google

Google's open-weight models. Strong on reasoning and factual tasks. Gemma 3 introduced multimodal support (text + image) in the 12B and 27B variants.

ModelNotes
gemma3:4bLightweight, surprisingly capable at 4B
gemma3:12bBest balance of quality and speed
gemma3:27bTop open-weight model at 27B, needs 20 GB RAM

Phi — Microsoft

The "small but smart" family. Phi models are optimised for reasoning quality per parameter — they outperform much larger models on coding and logical reasoning benchmarks.

ModelNotes
phi4:14bExceptional reasoning at 14B — beats many 70B models on maths/code
phi4-mini:3.8b3.8B with strong reasoning — great for 8 GB machines

If coding or maths reasoning is your primary use case and you have 16 GB RAM, phi4:14b is often the best local choice.

Qwen — Alibaba

Qwen 2.5 is one of the strongest open-weight model families in 2024–2025. Particularly strong on multilingual tasks and code generation.

ModelNotes
qwen2.5:7bStrong multilingual, competitive with Llama 3.1 8B
qwen2.5:14bExcellent code generation, 128k context
qwen2.5-coder:7bCode-specialised — competitive with DeepSeek Coder
qwq:32bReasoning model, think-before-answer like o1

Quick summary

  • Llama 3.1 8B — best all-rounder default, 128k context, 5 GB
  • Mistral 7B — fast and efficient, great for instruction following
  • Gemma 3 12B/27B — strong reasoning, multimodal on larger sizes
  • Phi-4 14B — best coding and maths reasoning at 16 GB RAM
  • Qwen 2.5 — best multilingual and code generation across sizes
